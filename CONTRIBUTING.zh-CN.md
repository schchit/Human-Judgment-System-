# 贡献指南

<p align="center">
  <strong>中文</strong> | <a href="CONTRIBUTING.md">English</a>
</p>

感谢您对 Human Judgment System (HJS) 项目的关注！我们欢迎所有形式的贡献。

## 🤝 如何贡献

### 1. 报告问题

如果您发现了问题或有改进建议，请：

1. 检查 [Issues](https://github.com/schchit/Human-Judgment-System/issues) 中是否已有相关问题
2. 如果没有，请创建新的 Issue，并详细描述问题或建议
3. 使用合适的标签（bug、enhancement、question等）

### 2. 提交代码

1. **Fork 仓库**
2. **创建分支**：`git checkout -b feature/your-feature-name`
3. **提交更改**：`git commit -m 'Add some feature'`
4. **推送分支**：`git push origin feature/your-feature-name`
5. **提交 Pull Request**

### 3. 改进文档

文档的改进同样重要！您可以：

- 修正拼写错误和语法问题
- 改进文档的清晰度和可读性
- 添加示例和用例
- 翻译文档到其他语言

### 4. 参与讨论

在 [GitHub Issues](https://github.com/schchit/Human-Judgment-System/issues) 中分享您的想法、提问或回答问题。

## 📝 代码规范

- 保持代码简洁和可读性
- 添加必要的注释
- 遵循项目的代码风格
- 确保提交信息清晰明确

## 🌍 多语言支持

我们支持中英文双语。如果您想添加其他语言的翻译，我们非常欢迎！

## ✅ 提交前检查清单

- [ ] 代码/文档已通过本地测试
- [ ] 遵循了项目的代码规范
- [ ] 更新了相关文档
- [ ] 提交信息清晰明确
- [ ] 没有引入不必要的依赖

---

再次感谢您的贡献！🎉
