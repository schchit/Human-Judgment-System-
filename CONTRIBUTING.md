Thank you for your interest in the Human Judgment System (HJS) project! We welcome all forms of contributions.

## ğŸ¤ How to Contribute

### 1. Report Issues

If you find a bug or have a suggestion, please:

1. Check if there's already a related issue in [Issues](https://github.com/schchit/Human-Judgment-System/issues)
2. If not, create a new Issue and describe the problem or suggestion in detail
3. Use appropriate labels (bug, enhancement, question, etc.)

### 2. Submit Code

1. **Fork the repository**
2. **Create a branch**: `git checkout -b feature/your-feature-name`
3. **Commit your changes**: `git commit -m 'Add some feature'`
4. **Push to the branch**: `git push origin feature/your-feature-name`
5. **Submit a Pull Request**

### 3. Improve Documentation

Documentation improvements are equally important! You can:

- Fix spelling errors and grammar issues
- Improve document clarity and readability
- Add examples and use cases
- Translate documents to other languages

### 4. Join Discussions

Share your thoughts, ask questions, or answer questions in [GitHub Issues](https://github.com/schchit/Human-Judgment-System/issues).
